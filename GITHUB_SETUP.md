### 3. Enable GitHub Pages

1. **Go to your repository**
2. **Click "Settings"** (top menu)
3. **Scroll down to "Pages"** (left sidebar)
4. **Source Settings:**
   - Source: "Deploy from a branch"
   - Branch: "main"
   - Folder: "/ (root)"
5. **Click "Save"**
